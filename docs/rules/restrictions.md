## 10. Ограничения (жёсткие)

- **Запрещено** использовать `useEffect` для фетчинга данных — для этого TanStack Query
- **Запрещено** передавать данные через prop drilling глубже 2 уровней — используй стор или контекст
- **Запрещено** писать бизнес-логику внутри JSX
- **Запрещено** импортировать из `mocks/` за пределами `main.tsx`
- **Запрещено** хардкодить URL в компонентах — только через константы в `shared/api/client.ts`
- Все публичные функции и хуки должны иметь JSDoc-комментарий минимум одной строкой
