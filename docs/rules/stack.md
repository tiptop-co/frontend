# Стек проекта

| Слой | Технология | Обоснование |
|---|---|---|
| Сборщик | **Vite** | Мгновенный старт, нулевая конфигурация, работает на Mac/Linux без боли |
| UI-фреймворк | **React 18** | Де-факто стандарт, огромная экосистема |
| Язык | **TypeScript** (strict) | Типы = живая документация контрактов для бекенда |
| Роутинг | **React Router v6** | Декларативный, file-system-агностичный |
| Серверный стейт | **TanStack Query v5** | Кэш, loading/error стейты, инвалидация — из коробки |
| Клиентский стейт | **Zustand** | Минималистичный, без бойлерплейта |
| Стили | **Tailwind CSS v3** | Utility-first, быстро при вайбкодинге, без конфликтов имён |
| Мок-слой | **MSW v2** (Mock Service Worker) | Перехватывает на уровне сети — компоненты не знают о моках |
| Линтер | **ESLint** + **Prettier** | Единый стиль, авто-форматирование |

**Запуск:**
```bash
npm create vite@latest my-app -- --template react-ts
cd my-app
npm install
npm run dev
```
> Никаких глобальных зависимостей. Только `node` и `npm`.
